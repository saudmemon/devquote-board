# MoodCompile 

A simple React app that shows random developer quotes, with the ability to like/save your favorites.

## Features

- Random quote generator
- Like/Dislike toggle with localStorage persistence
- Custom React Hook (`useQuote`) for clean logic separation
- Smooth fade-in animations
- Dark themed, card-based UI

## Built With

> React.js
> useState, useEffect Hooks
> Custom Hooks
> localStorage API
> CSS animations

## What I learned

This was my first hands-on React project. I focused on understanding core concepts deeply rather than just copying code:

- How state management works with `useState`
- Side effects and component lifecycle with `useEffect`
- Building reusable logic with custom hooks
- Persisting data in the browser with `localStorage`
